## 1. Slackに GenieSlack を追加
追加は以下のリンクから行えます。  
[Add to Slack ボタンへのリンク](https://genieslack.kusshi.dev/slack/install)

## 2. 初期設定
以下の操作は、Slackアプリ内でGenieSlackから送られる初期メッセージでも説明します。

### 2-1. Slackワークスペース上で、「要約して(summarize)」リアクションを作成する
1. Slackを開いて、メッセージ送信欄にある絵文字ボタンをクリックします。

![使い方画像 2-1 1](../images/how_to_use/howtouse_2-1-1.png '使い方画像 2-1 1')

2. ポップアップメニューから「絵文字を追加する」ボタンをクリックします。

![使い方画像 2-1 2](../images/how_to_use/howtouse_2-1-2.png '使い方画像 2-1 2')

3. 以下の画像をダウンロードします。 (画像を右クリックし、名前を付けて画像を保存)

![要約してスタンプ画像](../images/summarize.png '要約してスタンプ画像')

4. 「画像をアップロードする」ボタンを押し、ダウンロードした絵文字画像を指定します。
5. 「名前を付ける」で `:summarize:` を入力します。
6. 最後に「保存する」ボタンをクリックして、新たな絵文字の追加を完了します。

![使い方画像 2-1 4-6](../images/how_to_use/howtouse_2-1-4_6.png '使い方画像 2-1 4-6')

### 2-2. GenieSlack と esa を連携する
連携URLはGenieSlackからメッセージで送信されます。

1. SlackのダイレクトメッセージよりWebブラウザで連携URLにアクセスします。

![使い方画像2-2-1](../images/how_to_use/howtouse_2-2-1.png '使い方画像2-2-1')

2. GenieSlackから権限が要求されます。問題なければ承認してください。

![使い方画像2-2-2](../images/how_to_use/howtouse_2-2-2.png '使い方画像2-2-2')

3. ログインが要求されるので、ログインしてください。（ログイン済みの場合は省略されます）

![使い方画像2-2-3](../images/how_to_use/howtouse_2-2-3.png '使い方画像2-2-3')

4. 承認されると以下の画面が表示され、完了となります。

![使い方画像2-2-4](../images/how_to_use/howtouse_2-2-4.png '使い方画像2-2-4')


### 2-3.チャンネルにGenieSlackアプリを追加する。

1. Slackアプリ上でで右上のユーザアイコンをクリックしてください。

![使い方画像2-3-1](../images/how_to_use/howtouse_2-3-1.png '使い方画像2-3-1')

2. 「インテグレーション」タブを選択してください。

![使い方画像2-3-2](../images/how_to_use/howtouse_2-3-2.png '使い方画像2-3-2')

3. 「アプリを追加する」ボタンを押してください。

![使い方画像2-3-3](../images/how_to_use/howtouse_2-3-3.png '使い方画像2-3-3')

4. GenieSlackアプリの「追加」ボタンを押してください。

![使い方画像2-3-4](../images/how_to_use/howtouse_2-3-4.png '使い方画像2-3-4')

5. チャンネルにGenieSlackアプリが追加されました。

![使い方画像2-3-5](../images/how_to_use/howtouse_2-3-5.png '使い方画像2-3-5')

### 2-4. esaの記事を作成してカテゴリを作る

1. esaの `GenieSlack` カテゴリに、分類したいカテゴリ名で記事を作成してください。
    - デフォルトでは `GenieSlack/予定`, `GenieSlack/タスク`, `GenieSlack/Tips` が作成されます。
    - `GenieSlack/Tips/技術` のように、カテゴリを入れ子にすることもできます。
    - 最低でも1つのカテゴリを作ってください。

![使い方画像2-4-1](../images/how_to_use/howtouse_2-4-1.png '使い方画像2-4-1')

## 3. 利用時
1. 要約したいSlackのメッセージに、初期設定で作成した「要約」スタンプを付けます。
    - スタンプが見つからないときは、一覧から探してください。

![使い方画像 3 1](../images/how_to_use/howtouse_3-1.png '使い方画像3-1')

2. しばらくするとスタンプを押したメッセージのスレッドに、GenieSlackから返信が届きます。

![使い方画像 3 2](../images/how_to_use/howtouse_3-2.png '使い方画像3-2')

3. その返信に書かれているesaのURLから要約された内容を確認できます。

![使い方画像 3 3](../images/how_to_use/howtouse_3-3.png '使い方画像3-3')
