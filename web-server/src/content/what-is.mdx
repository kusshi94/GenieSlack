# **GenieSlack** - Slackの情報を簡潔に要約・カテゴライズ
GenieSlackは、Slackをさらに効果的に活用するためのアプリです。
Slackのチャット形式では、重要な情報が埋もれてしまうことがありませんか？
GenieSlackがその解決策となります。
![introduction](../images/what_is/introduction.png 'introduction')
  

---
    

# GenieSlackの主な機能
### 1. リアクションボタン１つでSlackのメッセージを要約
GenieSlackは、自然言語の生成系AIであるChatGPTを活用して、リアクションボタンを押すことでSlackのメッセージを要約します。
重要な情報や議論の要点を見逃すことなく、簡潔にまとめます。
そして、要約された情報はドキュメント共有サービスesaに自動的に書き込まれます。
![example1](../images/what_is/example1.png 'example1')

### 2. ChatGPTによる情報の要約・カテゴライズ
GenieSlackは、ChatGPTを活用して情報の要約・カテゴライズを行います。
重要なトピックや関連する情報を自動的に識別し、整理してくれます。  
これにより、迅速かつ効果的に必要な情報にアクセスすることができます。

### 3. esaで整理・要約された情報の閲覧
ドキュメント共有サービスesaに、GenieSlackが要約した情報が保存・整理されています。  
重要なトピックや議論の要点を見逃さず、簡単に参照することができます。
整理された情報を確認するだけでなく、他のチームメンバーと共有することもできます。

**重要な情報の見逃しや情報の散在を防ぎ、チーム全体のコラボレーションを強化できます。
